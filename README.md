# Proyecto Matriz Energética - Chile

**Autores:** Pablo Iracheta González y Franchesca Manríquez González

Este repositorio contiene los datos sintéticos y el código fuente utilizados para la sección de Metodología de nuestra investigación académica sobre la transición y matriz energética en Chile.

## Archivos incluidos en el repositorio:

* **`generador_datos.py`**: Script en Python diseñado para simular las tendencias históricas reportadas por fuentes gubernamentales (como el BNE). Genera datos longitudinales con variabilidad estadística para permitir inferencia.
* **`dataset_sintetico_energia.csv`**: Base de datos estructurada de 10 años resultante del script, que incluye métricas clave como el consumo de energía (GWh), el porcentaje de importación fósil y la capacidad solar instalada (MW) para Chile y una muestra de contraste.
